Porkbun static hosting upload instructions

1. Unzip this package.
2. Upload the CONTENTS of the cor_porkbun_site folder to your Porkbun static hosting web root.
3. Keep the folder structure intact, especially /assets and /documents.
4. The homepage is index.html.
5. Confirm these links work after upload:
   /documents/
   /timeline/
   /court-filings/

Included PDFs are in /documents.
